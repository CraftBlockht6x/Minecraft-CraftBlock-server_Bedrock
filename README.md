如何加入
要加入我们的服务器，请按照以下步骤操作：

打开我的世界基岩版。
点击“游戏”菜单。
选择“服务器”。
滑到最底下，点击“添加外部服务器”并输入服务器地址和端口，具体地址信息请在官网查找。
点击“加入”即可连接到服务器。
